# 🎮 Game Tebak Angka

**Mouza AI** adalah game web interaktif yang mencoba **membaca pola pikiran pemain** menggunakan sistem prediksi sederhana berbasis *learning pattern*.

Game ini menampilkan **AI assistant bernama Mouza** yang menganalisis angka yang kamu masukkan dan mencoba memprediksi angka berikutnya menggunakan pendekatan probabilitas dan pola perilaku.

## 🎮 Cara Bermain

1. Masukkan angka **1 sampai 10**
2. Klik tombol **INPUT**
3. AI Mouza akan:

   * Menganalisis pola pilihanmu
   * Menghitung probabilitas
   * Memberikan prediksi
4. Jika AI benar → **Mouza menang**
5. Jika AI salah → **Kamu berhasil mengecoh AI**

Game ini akan **belajar dari setiap angka yang kamu masukkan**.

---

## 🧠 Cara Kerja AI

Sistem prediksi menggunakan kombinasi beberapa metode sederhana:

### 1️⃣ Frequency Tracking

AI mencatat angka yang paling sering dipilih pemain.

### 2️⃣ Markov Transition

AI mempelajari pola transisi angka, misalnya:

```
jika pemain memilih 3
lalu biasanya memilih 7
```

AI akan meningkatkan kemungkinan prediksi angka tersebut.

### 3️⃣ Random Bias

Untuk menghindari prediksi yang terlalu deterministik, sistem menambahkan sedikit elemen acak.
